## 프로젝트의 목적
저는 이 프로젝트 통해 Spring Boot를 사용하여 RESTful API를 구현하는 것을 연습을 하기위해 만들었습니다. 

## 기술
* Java: JDK 21 <br>
* Spring Boot: 3.0<br>
* Frontend: Vue.js<br>
* IDE: IntelliJ IDEA, Visual Studio<br>
* Database: MySQL
